For Backend:
	First move to the Backend folder. Then Execute the following commands to run the Backend.
	(1) npm i nodemon 
	(2) npm i express
	(3) npm i cookie-parser
	(4) npm i bcrypt
	(5) npm i mongoose
	(6) npm i jsonwebtoken
	(7) npm i nodemailer
	(8) npm i axios
	(9) npm i dotenv
	(10) npm i cors
	(11) npm i morgan
	(12) node .\index.js

	
For Frontend:
	First move to the Frontend folder. Then Execute the following commands to run the Frontend.
	(1) npm install react-scripts
	(2) npm start
